### About

SentinelX is an independent cybersecurity research group focused on offensive security research, responsible vulnerability disclosure, and engineering security-grade tools and infrastructure.

We operate at the intersection of research and engineering — finding real vulnerabilities in real systems, and building production-grade software to solve real security problems.

---

### What We Do

- **Security Research** — Web application, API, and network vulnerability research across public and private sectors
- **Responsible Disclosure** — Coordinated disclosure with national cybersecurity agencies including BSSN
- **Security Engineering** — Developing custom daemons, automation frameworks, and hardened infrastructure from scratch
- **AI Integration** — Building AI-powered platforms tailored for security and developer workflows
